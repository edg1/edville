DESCRIPTION:
------------
This module provides custom webservices for druproid android appilcation .

REQUIREMENTS:
-------------
The module requires the services module.

INSTALLATION:
-------------
1. Place the entire services_extra directory into your Drupal sites/all/modules/
   directory.

2. Enable the services_extra module by navigating to:

     administer > modules

3. Enable the services by navigating to:

   administer > site building > services > custom service > edit Resources
